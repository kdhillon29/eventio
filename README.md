# Summary
Tech Summits and conferences events showcase and booking Web application developed in Nextjs,Supabase,drizzle-orm and typescript.

## Key features
+ Dynamic Events showcase using Nextjs partial pre-rendering feature with suspense.
+ Add new Events feature using shadcn and react-hook-form UI Compnents and Nextjs Api routes
+ Book Event using Server Actions 

## Install Packages

```bash
npm i
```

## Start the app

```bash
npm run dev
```
